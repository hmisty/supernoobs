# 学员名录

说明：
1. 刘教链超级小白课按期招生，一期一班，期号递增，称为：超级小白1期、超级小白2期、...（类似黄埔1期、黄埔2期、...）。
2. 超级小白班所有期次按报名顺序编以递增学号，学号终生有效，仅限本人使用。（发生未听课即转让名额的情况时，学号也会转让给他人，自己将不会被保留在名录上）
